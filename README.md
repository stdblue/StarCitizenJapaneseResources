# StarCitizenJapaneseResources

## Star Citizen v3.21.0 用非公式日本語リソース

　Star Citizen VERSION 3.20.0b より多言語リソース対応が CIG 公式からアナウンスされました。
　本リソースは英語メッセージを機械翻訳し、有志による手動翻訳をマージしたものとなります。

　翻訳作業は現在作業中であり、随時新しいリソースへと置き換えていきます。
